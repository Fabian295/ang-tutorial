  2. Property Binding

    Interpolation data binding is allways from "Child" to "Parrent", that means from "Component" to "Template" / View, if you will. Interpolation can only be used if the binding value is a "String".

    If the binding value is not a String we use "Property Bonding. We put the attribute in square brackets, i.e.: [src]="imageUrl". The component.tss file variable declaration ( with the path/to/the/img ) stays the same.
