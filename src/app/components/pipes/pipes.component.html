What are pipes

If you have worked with AngularJS it's filters, then it wil be easy for you. These filters are known as "Pipes" in Angular. For every body else: Pipes allow us to change the data before displaying it. Normally a pipe takes this data and transforms this input to the desired output.

Syntax of pipes

Expression : is the expression which you want to transfor

| : Pipe operator's Symbol / charactesr

pipeArguments : arguments to the Pipe

Custom Pipes

If the built-in pipes are not specific enough or do not result in a satisfying solution, then you are free to generate and customize your own pipe.

You can use the cli to do this. Just type
ng g p dir/pipe_name The g and p stands for "generate pipe" and will create two files.

  1. pipe_name.pipe.spec.ts
  2. pipe_name.pipe.ts
The first file is for testing, the second file is a class.

Custom Pipes

Angular cli generated code for the .pipe.ts file. A class with the @Pipe decorator and the pre-defined interface "PipeTransform" implemented which comes with the transform method.
